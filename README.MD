## Como baixar o projeto

```bash
    
    # Clonar o repositório
    $ git clone https://github.com/eugenio-silva/Next-Level-Week-04.git
    
    # Entrar no diretório
    $ cd Next-Level-Week-04
    
    # Instalar as dependências
    $ yarn install
    
    # Iniciar o projeto
    $ yarn dev
    
```
